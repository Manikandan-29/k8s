```

## Create IAM role
```=20

  --cluster=3D&lt;your-cluster-name&gt; \
  --namespace=3Dkube-system \
  --name=3Daws-load-balancer-controller \
  --role-name AmazonEKSLoadBalancerControllerRole \
  --attach-policy-arn=3Darn:aws:iam::&lt;your-aws-account-id&gt;:policy/AWS=
LoadBalancerControllerIAMPolicy \
  --approve
```
